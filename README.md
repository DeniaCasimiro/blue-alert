# blue-alert

Minimal Vite + React scaffold showing a blue car with an "Alert" speech bubble.

Getting started

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

Build

```bash
npm run build
npm run preview
```

Notes

- `node_modules/` is ignored via `.gitignore`.
- The working app is in `src/App.jsx` and the entry is `src/main.jsx`.
- Static image `blue-car.png` is located at project root and imported from `src/App.jsx`.
